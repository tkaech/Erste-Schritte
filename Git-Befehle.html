Nachfolgend ein paar Git-Befehle

git init //erstellt ein neues Repository git add . //fügt alle Dateien dem Repository hinzu git add Git-Befehle.html //fügt nur die genannte Datei hinzug git status // Zeigt den aktuellen Status an git commit -m "add Git-Befehle" git log //Listet die Commits auf

Diff-Befehl

git diff .\Git-Befehle.html //Unterschied zur angegebenen Datei. Zuvor speichern, es wird nur die gespeicherte Datei verglichen und nicht die im Memory geändert.
